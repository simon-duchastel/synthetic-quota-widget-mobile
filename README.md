# synthetic-quota-widget-mobile
A small widget for your Android home screen showing how much of your [Synthetic.new](https://synthetic.new) subscription you've used

## Architecture Notes

### Widget ID vs Glance ID

This app uses two different identifiers:

- **Glance ID**: A system-assigned identifier from Android's Glance framework. Each widget instance has a unique Glance ID, but these IDs can change between app restarts.
- **Widget ID**: A UUID generated by us and stored in the widget's state. This ID persists across app restarts and is used to identify specific widget instances when syncing data.

When a refresh is triggered:
1. The widget ID is passed to the sync worker
2. The worker only updates widgets whose stored widget ID matches the passed ID
3. If no widget ID is passed (e.g., scheduled sync), all widgets are updated

This ensures that manual refreshes only affect the widget that was clicked, while periodic syncs update all widgets.
